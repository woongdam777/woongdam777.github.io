## 빌드란 뭘까?

- 빌드란 다음과 같은 동작을 수행하는 일을 말한다
  - 소스 코드를 컴파일
    - 필요한 라이브러리를 다운 받고 \*classpath에 추가
  - 테스트를 실행
  - 컴파일된 코드를 패키지(jar, war, zip 등)
  - 패지키된 파일을 주로 artifact라고 부르고 서버나 레포지토리에 배포
  - classpath : 사용자 정의 클래스 및 API 패키지의 경로를 지정하는 자바가상머신 또는 Java 컴파일러의 매개 변수
